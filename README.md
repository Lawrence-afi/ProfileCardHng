# 📌 Profile Card UI

A simple and responsive profile card built with **HTML**, **Tailwind CSS**, **Font Awesome**, and **Vanilla JavaScript**. It displays a user profile, bio, hobbies, dislikes, social links, and a dynamic time display.

---

## 🚀 Features

- Responsive centered profile card layout
- Circular profile image with shadow styling
- User bio section
- Hobbies and dislikes tags
- Gradient call-to-action button
- Social media icons (GitHub, X, LinkedIn, Facebook)
- Live time display using JavaScript

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- Font Awesome
- JavaScript (Vanilla)
